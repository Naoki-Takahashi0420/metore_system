# よくある質問（FAQ）

## ログイン・アクセス

### ログインできません
1. メールアドレスとパスワードを確認してください
2. パスワードを忘れた場合は「パスワードリセット」リンクから再設定
3. それでも解決しない場合は管理者（スーパー管理者）に連絡してパスワードをリセットしてもらってください

### 権限エラーが表示されます
- あなたのアカウント権限では、その機能にアクセスできません
- 必要な場合は管理者に権限変更を依頼してください
- 各権限でできることは以下の通り：
  - **super_admin**: 全機能アクセス可能
  - **owner**: 複数店舗管理、売上レポート確認
  - **store_manager**: 所属店舗の全機能、スタッフ管理
  - **staff**: 所属店舗の予約・カルテ作成のみ

## 予約関連

### タイムラインから予約を作りたい
1. ダッシュボードの「予約タイムライン」を表示
2. 空き時間をクリック
3. 顧客を検索・選択
4. メニューを選択
5. 開始時間と所要時間を調整
6. 「予約を作成」ボタンをクリック

**💡 ポイント**: タイムラインなら空き状況を確認しながら予約を作れます

### 予約が重複してしまいました
1. 予約管理画面で重複している予約を確認
2. 不要な方を選択して「編集」→ステータスを「キャンセル」に変更
3. 「保存」ボタンをクリック

### 予約通知が顧客に届きません
1. 顧客のLINE連携状況を確認（顧客詳細画面）
2. LINE未連携の場合、SMS通知が送信されているか確認
3. 電話番号が正しいか確認（ハイフンなし）
4. 設定→通知設定で通知が有効になっているか確認

### 予約時間を変更したい
- **方法1**: 予約詳細画面で「編集」→日時を変更
- **方法2**: タイムライン上で予約をドラッグ&ドロップ

### サブスク/チケット利用の予約はどうやって作る？
1. 通常の予約作成手順と同じ
2. 顧客選択時に、サブスク/チケット保有者を選択
3. メニュー選択時に「サブスクメニュー」または「チケット利用」を選択
4. 自動的に回数が消費されます

## 顧客管理

### 顧客情報を間違えて登録してしまいました
1. 顧客管理画面で該当顧客を選択
2. 「編集」ボタンをクリック
3. 正しい情報に修正
4. 「保存」ボタンをクリック

### 重複した顧客を統合したい
- 現在、自動統合機能はありません
- 正しい顧客情報を残し、重複データは削除してください
- **⚠️ 注意**: 予約履歴は移行できません

### 顧客の電話番号を変更したい
1. 顧客編集画面で電話番号を変更
2. LINE連携には影響しません
3. 「保存」ボタンをクリック

### カルテはどこで作成しますか？
1. 顧客詳細画面を開く
2. 「カルテ」タブをクリック
3. 「新規カルテ作成」ボタンをクリック
4. 施術内容、視力測定結果などを入力
5. 「保存」ボタンをクリック

### カルテが保存できません
- 必須項目（視力測定値）が入力されているか確認
- ブラウザをリロードして再度お試しください

### 視力推移グラフが表示されません
- 複数回のカルテ記録が必要です
- 最低2回以上の測定記録があるか確認

## メニュー・カテゴリ

### メニューを作成したい
1. サイドバー「メニュー管理」→「メニュー」をクリック
2. 右上の「新規作成」ボタンをクリック
3. 基本情報を入力：
   - 店舗
   - メニュー名
   - カテゴリー（先に店舗を選択してください）
   - 料金
   - 所要時間
4. 「作成」ボタンをクリック

### サブスクメニューと通常メニューの違いは？
- **サブスクメニュー**: 月額制で、月間利用回数内で何度でも予約可能
- **通常メニュー**: 都度払いで、1回ごとに料金が発生
- **作成方法**: メニュー作成時に「サブスクリプションメニューとして提供」をON/OFFで切り替え

### オプションメニューだけ予約できますか？
いいえ、オプションメニューは必ずメインメニューと一緒に予約する必要があります。
メイン施術に追加するサービスとして使用してください。

### メニューの表示順を変えたい
1. メニュー一覧で対象メニューをクリック
2. 「編集」ボタンをクリック
3. 「表示順」の数字を変更（小さいほど上位）
4. 「保存」ボタンをクリック

### カテゴリを削除できません
- そのカテゴリに紐づくメニューがある場合は削除できません
- 先にメニューを別カテゴリに移動するか削除してください
- または、カテゴリを「無効化」することで非表示にできます

### メニューが選択できません
- 先に店舗を選択してください
- 店舗を選択するとその店舗のメニューのみが表示されます

## サブスク・チケット

### サブスク顧客の利用回数が足りなくなりました
1. サイドバー「サブスク管理」→「顧客サブスク」
2. 該当契約を選択
3. 「編集」ボタンをクリック
4. 「月間利用回数」を増やす
5. 「保存」ボタンをクリック

### チケットの有効期限を延長したい
1. サイドバー「チケット管理」→「顧客チケット」
2. 該当チケットを選択
3. 「編集」ボタンをクリック
4. 「有効期限」を変更
5. 延長理由を記入
6. 「保存」ボタンをクリック

### サブスク予約をキャンセルしたら回数は戻りますか？
はい、予約をキャンセルすると自動的に利用回数が復元されます。

### チケット予約をキャンセルしたら回数は戻りますか？
はい、予約をキャンセルすると自動的に残回数が復元されます。

### チケットとサブスクを同時に契約できますか？
はい、同じ顧客が両方を契約できます。予約時にどちらを使うか選択できます。

### サブスクの利用回数は翌月に繰り越せますか？
いいえ、繰越はできません。毎月1日に自動的にリセットされます。

## シフト管理

### シフトを登録していないスタッフで予約できますか？
できますが、警告が表示されます。シフト外のスタッフは予約選択肢に表示されません。

### 繰り返しシフトを一括登録したい
1. シフト管理画面で「一括登録」ボタンをクリック
2. 対象スタッフを選択
3. 期間を指定（開始日〜終了日）
4. 曜日を選択（複数選択可）
5. 開始時刻・終了時刻を入力
6. 「一括作成」ボタンをクリック

### シフト変更したら予約はどうなりますか？
- 既存の予約には影響しません
- ただし、シフト削除時に予約があると警告が表示されます
- 予約を先にキャンセル/変更してからシフトを削除してください

### 休憩時間中に予約を入れたくない
1. シフト登録時に休憩時間を設定
2. タイムライン上でグレー表示され、自動的に予約不可になります

## システム全般

### 動作が遅い
1. ブラウザのキャッシュをクリア
2. Ctrl+Shift+R（Mac: Cmd+Shift+R）で強制リロード
3. 別のブラウザで試す（推奨: Chrome, Edge）
4. インターネット接続を確認
5. 設定→システムから「キャッシュクリア」を実行

### エラーが表示されます
1. ページをリロード（F5キー）
2. 一度ログアウトして再ログイン
3. 解決しない場合はエラーメッセージのスクリーンショットを撮って管理者に連絡

### 画面が真っ白になります
1. ブラウザのキャッシュをクリア
2. Ctrl+Shift+R（Mac: Cmd+Shift+R）で強制リロード
3. 別のブラウザで試す
4. それでも解決しない場合は管理者に連絡

### 右下の?ボタンは何ですか？
- **AIヘルプチャット**です
- クリックするとシステムの使い方を質問できます
- システムマニュアルを参照して回答します
- わからないことがあればお気軽にご利用ください

### ヘルプチャットで解決しなかった場合は？
1. 回答の下にある「解決しませんでした」ボタンを押す
2. フィードバック内容を入力
3. 「送信」ボタンをクリック
4. 管理者が確認して対応します

### スタッフの権限を変更したい
1. サイドバー「設定」→「ユーザー管理」
2. 対象スタッフを選択
3. 「編集」ボタンをクリック
4. 「権限」を変更
5. 「保存」ボタンをクリック
**⚠️ 注意**: 店長以上の権限が必要です

### データのバックアップはどうすれば？
1. サイドバー「設定」→「バックアップ」
2. 「今すぐバックアップ」ボタンをクリック
3. ファイルがダウンロードされます
**推奨**: 毎日自動バックアップ + 週1回手動確認

### 通知設定はどこで変更できますか？
1. サイドバー「設定」→「通知設定」
2. LINE/メール/SMS通知をそれぞれ設定
3. テスト送信で動作確認
4. 「保存」ボタンをクリック

## トラブルシューティング

### 予約が消えてしまいました
1. 誤ってキャンセルした可能性があります
2. 予約管理画面のフィルターで「キャンセル」を選択
3. 該当予約を見つけたら、ステータスを「確定」に戻す

### タイムラインに予約が表示されません
1. 店舗と日付のフィルターを確認
2. ブラウザをリロード（F5キー）
3. キャッシュをクリア

### 顧客が検索で見つかりません
- 姓または名で部分一致検索を試してください
- 電話番号でも検索可能です
- フリガナでも検索できます

### カルテの印刷ができません
1. ブラウザの印刷設定を確認
2. PDFとして保存してから印刷
3. 別のブラウザで試す

### LINE通知が届きません
1. 顧客がLINE連携しているか確認
2. 顧客がLINEをブロックしていないか確認
3. 設定→通知設定でLINE設定を確認
4. 「接続テスト」ボタンで動作確認
